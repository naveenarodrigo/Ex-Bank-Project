# Ex-Bank Banking Management System

## Overview
Ex-Bank is a banking management system developed using Java Enterprise technologies.

## Features
- User Registration
- User Login
- Account Management
- Deposit Money
- Withdraw Money
- Transfer Funds
- Transaction History

## Technologies Used
- Java
- EJB
- JSP
- HTML
- CSS
- JavaScript
- Maven

## Project Structure
- core
- ejb
- web
- ear
